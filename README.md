#HiveBoard
[昆虫棋](https://zh.wikipedia.org/zh/%E6%98%86%E8%9F%B2%E6%A3%8B)是John Yianni在2001年推出，以昆蟲為主題，兩人對戰的棋類遊戲，機制類似塊相棋。

##技术架构

####前端:
1. Electron
2. React
3. Rx(暂定)
4. ES6
5. socket.io
6. babel
7. webpack

`npm install 安装所有依赖`

`npm run app 启动Electron`

####后端:
1. koa
2. socket.io

`npm run start 启动服务器`
